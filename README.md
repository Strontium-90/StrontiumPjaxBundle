StrontiumPjaxBundle
===================


Installation
------------

``` json
{
    "require": {
        "components/jquery-pjax": "*"
    }
}
```

To start using the bundle, register it in your Kernel:

``` php
<?php
// app/AppKernel.php

public function registerBundles()
{
    $bundles = array(
        // ...
        new Strontium\PjaxBundle\StrontiumPjaxBundle(),
    );
    // ...
}
```
